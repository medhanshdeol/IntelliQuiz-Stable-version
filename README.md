# IntelliQuiz(Stable Version) 

IntelliQuiz is an intelligent, LLM-powered platform that transforms any topic into an interactive learning experience. Beyond just generating quizzes, it features a custom **ML Scratchpad** where core classification algorithms are implemented from scratch to provide transparency into how AI makes decisions.

Key Features

* **AI-Powered Generation:** Uses LLMs to dynamically generate quizzes and synthetic datasets based on user-defined topics.
* **ML Scratchpad:** A dedicated module for exploring data science fundamentals without external ML libraries.
* **Full-Stack Architecture:** A seamless integration of a modern frontend, robust backend, and AI orchestration.

The ML Engine (Built from Scratch)

The heart of this project is the manual implementation of foundational machine learning algorithms. This allows users to see the mathematical logic behind the predictions.

Implemented Algorithms:
* **Decision Tree Metrics:** * **Entropy:** Measuring dataset impurity.
    * **Information Gain (ID3):** Identifying the most descriptive features.
    * **Gain Ratio (C4.5):** An improved approach to handle feature biasing.
* **Probabilistic Classification:** * **Naive Bayes:** Implementing Bayes' Theorem for predictive modeling.
    * **Laplace Smoothing:** Ensuring the model handles unseen data points gracefully without zero-probability errors.
